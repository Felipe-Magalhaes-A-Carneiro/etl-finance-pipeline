# Pipeline de Ingestão e ETL de Dados Financeiros em Tempo Real

Este projeto simula um pipeline de dados corporativo que automatiza a coleta de indicadores macroeconômicos para alimentar painéis de tomada de decisão financeira.

[ API Externa ] ---> ( Script Python / Requests ) ---> [ Limpeza / Pandas ] ---> [ Banco PostgreSQL ]

## Tecnologias Utilizadas:
Python 3, Pandas, SQLAlchemy, SQLite/PostgreSQL, Git

## Como Executar:

pip install -r requirements.txt e python main.py
